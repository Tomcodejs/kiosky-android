Repo officiel de l'application Android Kiosky Launcher (com.kiosky.launcher)
----
Kiosky Launcher est une application déstinée au téléphones portables tournant sous Android (OneUI, crDroid, Lineage etc...) pour les transformer en sorte de StreamDeck pour lancer des applications avec des boutons personnalisables.

Le repo est mis à jour régulièrement, donc vérifie régulièrement pour avoir la dernière version de Kiosky Launcher !

Comment mettre l'application en launcher par défaut ? Dans les paramètres : Applications -> Applis par défaut -> Appli d'accueil -> Kiosky (En tout cas ça marche sur crDroid)
